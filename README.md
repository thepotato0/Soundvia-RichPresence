# Soundvia-RichPresence
Python integration of Soundvia and Discord RPC

# Requirements

- Python 3.8+
- Discord desktop app running
- Soundvia app and token

# Installation
> Install dependencies
```bash
pip install -r requirements.txt
```
# Usage
> Run the app with either commands
```bash
python main.py
```
```bash
python3 main.py
```
